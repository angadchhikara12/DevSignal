# DevSignal

DevSignal is a lightweight, frontend-only developer analytics tool that evaluates coding profiles using public GitHub activity signals.

It analyzes repositories, commits, structure, and consistency to generate a transparent engineering profile and “Developer Signal Score.”

> Not a hiring predictor — a skill reflection tool.

---

## 🚀 What It Does

- Analyzes GitHub profiles using public data
- Computes a structured Developer Signal Score
- Breaks down strengths across:
  - Code Quality
  - Consistency
  - Project Depth
  - Engineering Maturity
  - Collaboration Signals
- Detects weak areas in your development profile
- Provides actionable improvement suggestions

---

## 🧠 Core Idea

Instead of guessing or relying on self-reported data, DevSignal uses real engineering signals from GitHub activity to build an honest snapshot of developer growth and skill direction.

---

## 📊 Example Output

- Developer Type: Builder Developer  
- Signal Score: 72%  
- Strength: Project Depth  
- Weak Spot: Inconsistent Activity  

---

## ⚙️ Tech Stack

- Vanilla JavaScript
- HTML + CSS
- GitHub REST API

No backend. No frameworks. No login required.

---

## 📌 Limitations

- Uses only public GitHub data
- Does not access private repositories
- Does not guarantee hiring outcomes
- No external platform integrations (frontend-only)

---

## 🎯 Goal

To help developers understand:
- What they’re good at
- What they’re missing
- How their GitHub activity reflects real engineering skill
